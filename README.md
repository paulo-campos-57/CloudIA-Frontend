## About

This is the CloudIA frontend, a web application that allows users to ask questions about Cloud Computing and receive answers generated by the CloudIA API, which uses Google's Gemini 2.5 Flash model.

### Features

- Intuitive and responsive interface
- Query form with response size selection
- Formatted display of CloudIA responses
- Universe-inspired design with cool color palette
- Smooth animations and transitions
- Automatic retry system for requests
- Project about page and contact section
- Fully responsive (mobile, tablet, desktop)

## Local Installation

### Prerequisites

- Node.js 18.x or higher
- npm, yarn or pnpm

### Steps

1. Clone the repository:
```bash

cd frontend-cloudia
```

2. Install dependencies:
```bash
npm install
```

3. Configure the API URL:

Edit the `src/pages/Index.tsx` file and change the `API_URL` constant:
```typescript
const API_URL = "https://cloudia-ljca.onrender.com/api/assistente/explicar";
```

4. Start the development server:

## How It Works

### Usage Flow

1. User accesses the home page
2. Enters a topic about Cloud Computing
3. Selects response size (short, medium, long)
4. Clicks "Ask CloudIA"
5. System sends request to CloudIA API
6. Response is displayed in formatted way on screen

### API Configuration

```typescript
// Payload sent to API
{
  topic: "Cloud Computing Topic",
  tipo_resposta: "m√©dia" // "curta" | "m√©dia" | "longa"
}

// Expected API response
{
  explanation: "Response generated by Gemini..."
}
```

### Retry System

The frontend implements automatic retry with exponential backoff:
- Maximum of 20 attempts
- Exponential delay: 2^attempt * 1000ms
- Detailed logs in console

## Troubleshooting

### CORS Error
- Check if CloudIA API allows requests from your domain
- Configure CORS on backend if needed

### API not responding
- Verify API URL is correct
- Render's Free plan hibernates after inactivity
- Wait for cold start (20-30 seconds)

### Styles not applied
- Run `npm run build` again
- Clear browser cache
- Verify Tailwind is configured correctly
